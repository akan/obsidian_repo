# 智能代理回到未来 - 创意手绘风PPT

## 📋 项目说明

本项目基于ASCII框架文件自动生成了一套创意手绘风格的PowerPoint演示文稿，主题为"智能代理回到未来 - AI架构演进与实践探索"。

## 📁 文件结构

```
智能代理回到未来_创意手绘风PPT.pptx          # 完整PPT文件 (3.4MB)
PPT预览页面/                                 # 预览页面目录
├── index.html                              # 预览首页
├── slide_01.html                           # 第1张幻灯片 (封面)
├── slide_02.html                           # 第2张幻灯片 (目录)
├── ...                                     # 其他幻灯片
└── slide_13.html                           # 第13张幻灯片 (结束)
```

## 🎨 设计特色

- **创意手绘风格**: 采用手绘、水彩风格的视觉设计
- **柔和配色**: 使用温暖的米色、柔和蓝色、珊瑚粉和薄荷绿
- **装饰元素**: 添加手绘风格的边框、箭头和装饰图案
- **优雅字体**: 结合中文书法字体和现代无衬线字体
- **响应式布局**: 适配不同屏幕尺寸

## 💻 使用方法

### 方法一：下载PowerPoint文件
1. 下载 `智能代理回到未来_创意手绘风PPT.pptx`
2. 使用Microsoft PowerPoint、WPS或Google Slides打开
3. 进行编辑或演示

### 方法二：浏览器预览
1. 打开 `PPT预览页面/index.html`
2. 在浏览器中查看所有幻灯片
3. 点击任意卡片可全屏查看单张幻灯片
4. 使用方向键（←→↑↓）快速浏览

## 🛠️ 技术实现

本PPT通过以下技术栈生成：

- **HTML5 + CSS3**: 幻灯片内容与样式
- **JavaScript**: 交互功能
- **Playwright**: HTML到图片转换
- **PptxGenJS**: 生成PowerPoint文件

## 📝 自定义修改

如需修改内容：

1. **编辑HTML文件**: 直接修改 `PPT预览页面/slide_XX.html`
2. **重新生成PPT**: 运行转换脚本
3. **调整样式**: 修改CSS样式变量
