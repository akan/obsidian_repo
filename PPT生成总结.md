## 🛠️ 技术实现

### 生成流程
```
ASCII框架 → HTML/CSS生成 → 浏览器截图 → PPTX转换 → 最终输出
    ↓           ↓            ↓           ↓
  内容解析   样式设计    Playwright   PptxGenJS
```

### 核心技术栈
- **前端**: HTML5, CSS3, JavaScript
- **截图**: Playwright (无头浏览器)
- **生成**: PptxGenJS (PPT生成库)
- **样式**: 响应式设计, Flexbox/Grid
