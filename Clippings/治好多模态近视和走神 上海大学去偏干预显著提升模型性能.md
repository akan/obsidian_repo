多模态大模型在观察世界时其实存在严重的近视和走神。

它们总是过度关注图像底部而忽略核心内容。

上海大学与南开大学的研究团队，仅仅通过两条数学公式的去偏干预，就能在不增加计算成本的前提下显著提升所有主流模型的视觉理解能力。

![Image](https://mmbiz.qpic.cn/mmbiz_png/FMjxhjKQXwy56jzE4ozQcKKiaKJdAJViceY1ribDOW7NRtYMBYqBia4GicIEWWYHBe00KyDnJEEx2KibZYf3pIAicibPnY21KUUWFfp2X6licvmUHfMI/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)

视觉语言模型（VLM）在处理图像时极其奢侈，需要成百上千个视觉 Token 来描述一张图，这种冗余导致推理效率低下。

学术界广泛采用注意力机制来筛选重要的视觉信号，也就是所谓的 Token Pruning（Token 剪枝），通过保留注意力分数高的部分来减负。

研究团队发现，这种看似科学的筛选标准其实带有严重的系统性偏见。

模型天生更喜欢序列末尾的信号，甚至会盯着毫无意义的空白填充区域发呆。

而他们提出的注意力去偏技术，像一把精准的手术刀，切除了算法中遗传自大语言模型的认知顽疾，让视觉筛选重新回归语义本身。

### 注意力分数背后的位置陷阱

大语言模型在训练过程中养成了一些古怪的习惯，其中最明显的就是 Recency Bias（近因偏差）。

这种现象在人类心理学中也很常见，我们往往对刚刚发生的事情记忆犹新，而模型则表现为对序列后面出现的 Token 给予不成比例的高关注度。

当这种特性被带入到视觉领域时，问题变得滑稽起来。

因为视觉 Token 是按顺序排列的，图像底部的像素往往排在序列末尾，导致模型总觉得图片底部更重要。

模型在观察一张风景画时，可能仅仅因为草地排在序列最后，就给草地分配了极高的权重，反而忽略了图像中间那架划破长空的飞机。

这种位置上的先天优势掩盖了内容本身的价值，使得剪枝算法在精简信息时，往往误删了真正的视觉核心。

除了位置偏见，模型还患有严重的 Attention Sink（注意力陷阱）综合征。

它们会莫名其妙地在序列开头的 BOS（起始 Token）或者为了对齐长度而加入的 Padding（填充 Token）上投入大量注意力。

这些填充 Token 本该是透明的背景噪音，却因为模型架构的缺陷变成了吸金黑洞，夺走了本属于关键物体的光芒。

研究人员通过大量的实验观察发现，这种偏差是系统性的且普遍存在于各种主流架构中。

如果不修正这些偏见，任何基于注意力的剪枝方法都像是在一面哈哈镜里寻找最美的风景，得到的永远是扭曲的结果。

下图是六种不同视觉剪枝方法在十个图像视觉问答基准测试中的平均表现。

![Image](https://mmbiz.qpic.cn/sz_mmbiz_png/FMjxhjKQXwy0EsyBtPWIiceTuydSIhHaY8GbPsCLIibNyTlwV63PQpsUZWv3F8iand1uO8ibmz1GIsemFIfd3uQibsmsxfoxWiaAH5kWMmEbSYuIo/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

每个顶点代表一种剪枝方法，橙色线条代表加入去偏技术后的表现。可以清晰看到，去偏后在所有六种现有的剪枝模型上都实现了稳定的性能跨越。

### 数学曲线还原真实的视觉价值

为了对抗顽固的近因偏差，研究团队设计了一套极其轻量化的数学修正工具。

他们首先统计了模型在处理海量数据时，注意力分数随位置变化的自然分布规律。

通过对这些数据进行拟合，研究者发现这种位置偏好完美符合一种指数增长趋势。

通过一个简单的指数函数来描述这种天生的不公平性，并将其作为分母来抵消原始分数中的位置溢价。

这种处理方式非常巧妙，它将原本复杂的认知偏差抽象成了一条平滑的曲线。

当我们用原始的注意力分数除以这个位置增益系数，得到的就是去掉了位置滤镜后的真金白银。

这就像是在一场长跑比赛中，给起跑线靠后的选手减去了一定的物理距离加成，从而让所有人都在同一条语义起跑线上竞争。

经过这种归一化处理，模型终于能够平视图片中的每一个角落，不再盲目崇拜那些恰好出现在序列末尾的幸运儿。

这种方法的通用性极强，因为它不依赖于特定的任务或者模型参数，几乎可以无缝嵌入到任何现有的 VLM 推理流程中。

研究者们称之为 Positional Debiasing（位置去偏），它本质上是赋予了模型一种空间正义感。

在这种新标准的审视下，那些真正对回答问题有帮助的视觉特征，无论它在图片的顶部、中间还是底部，都能凭借自身实力脱颖而出。

![Image](https://mmbiz.qpic.cn/sz_mmbiz_png/FMjxhjKQXwzL6StoJzyPh4XCczyOIveZsjwvtk5nGnfa7afZMy6RgaRBjua6wia6Kcw6Bvic8HC9Ue1jH5nwWDmOVOFL7wVZeI4RicgKdjMhVU/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)

LLaVA-v1.5-7B 模型中原始注意力分数与去偏后的对比：原始曲线随 Token 索引剧增，显示出强烈的近因偏差；蓝色曲线在应用指数拟合去偏后，表现出更加稳定且与内容相关的分布特性。

### 零成本消除填充区域的幻觉

解决了位置偏差后，剩下的硬骨头就是那些由于工程需要而存在的填充区域。

在处理不同尺寸的图像时，为了让计算矩阵整齐划一，系统往往会在图片边缘补上一些毫无意义的 Padding（填充 Token）。

由于 Attention Sink（注意力陷阱）效应的存在，模型会给这些空白地带分配极高的分数。

这在剪枝过程中造成了灾难性的后果，算法会小心翼翼地保留这些空白，却把含有关键信息的像素当成垃圾扔掉。

研究团队采取了一种极为直接且有效的策略，即 Padding Attention Suppression（填充注意力抑制）。

既然我们明确知道哪些区域是人为添加的垃圾信号，那么在计算重要性排名时，直接将这些区域的注意力分数手动归零。

这种操作就像是在考卷上直接划掉无关紧要的涂鸦，强制让模型的目光聚焦在真实的视觉内容上。

这种逻辑虽然简单，但在实际应用中却爆发出了惊人的威力。

通过将位置去偏与填充抑制相结合，研究团队构建了一个双层防护网。

第一层滤网去掉了位置带来的虚假繁荣，第二层滤网排干了填充区域带来的冗余水汽。

最终呈现在剪枝算法面前的是一份干净、真实且完全基于语义逻辑的贡献度清单。

![Image](https://mmbiz.qpic.cn/sz_mmbiz_png/FMjxhjKQXwwbcSBhxibNI0mFm027bV0e0ia2WAeZR0bBLPiaDgG180JQf3EqMQibFkNamHIBc0rHJ7m8INKO0bIhS3iaP6ztx7SnuLWEcfbN4Pfk/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)

这种去偏后的注意力信号不仅更加准确，而且具备了极强的逻辑说服力，真正实现了模型推理效率与准确性的双重飞跃。

### 跨模态实验验证了去偏的广度

研究人员在十个图像基准测试和三个视频基准测试中进行了地毯式的压力测试。

这些测试涵盖了从简单的物体识别到复杂的逻辑推理，甚至是针对视障人士真实拍摄的模糊照片。

![Image](https://mmbiz.qpic.cn/mmbiz_png/FMjxhjKQXwwXtCzL7gYBYvJeNWUXPz8xWuOqtoB1qrmwGT5TbKb7fSukXWWuQBhiaiczQck7wLgMMnqXe74TaqIA5QicOkFUALI11fUb2ibAKuA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=4)

实验结果显示，无论是在 7B 还是 13B 参数规模的模型上，这套去偏技术都带来了显著的性能提升。

尤其是在那些对细节要求极高的视觉问答任务中，去偏后的模型表现得更加聪明且专注。

在视频理解任务中，由于视频帧产生的 Token 数量更为庞大，冗余问题也更加突出。

研究团队将方法引入到 Video-LLaVA（视频视觉语言模型）框架中，在处理长达 2048 个 Token 的视频序列时，即便强制剪掉四分之三的信号，去偏后的模型依然能保持极高的回答准确度。

证明了近因偏差不仅存在于二维的静态图像中，也存在于三维的时间序列中。

更有趣的是，这种改进完全不需要重新训练任何参数，它更像是一种思维方式的纠偏。

研究人员还对比了修改位置编码方案（如 RoPE 旋转位置编码）的方法，发现那种从底层架构大动干戈的方案，效果反而不如这种在推理阶段进行的简单数学干预。

这反映了一个深刻的技术洞察，即大模型的偏差往往源于训练目标的本质，而非单一的编码技术。

### 视觉可视化揭示了选择的逻辑

为了让去偏效果更加直观，研究团队展示了剪枝后的像素分布热图。

![Image](https://mmbiz.qpic.cn/mmbiz_png/FMjxhjKQXwxiaL27Jx3icYq9n2pTVTcFEUQckj3yabicfuPgJwg3K7xMmr6EWkjA92Um7AKUbMu2icvbhZ0JF4z57YKG1cQX9h1VYiay1SxEWPbU/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=5)

在传统的 FastV（快速视觉）算法下，模型在处理不同比例的图片时，选择的 Token 分布极其诡异，经常成片地保留图片底部或边缘的填充黑条。

这解释了为什么之前的模型在处理长图或宽图时经常翻车，因为它们把宝贵的算力都浪费在观察背景板上了。

应用去偏技术后，热图发生了翻天覆地的变化。

原本集中在底部的亮点散开到了图像中有意义的物体上，这种分布变得更加均匀且符合人类的直觉。

在处理一张穿着队服的棒球运动员照片时，原始算法只盯着运动员的脚踝和身后的草地看，而新算法则精准地保留了运动员背上的号码和球衣上的标志。

在处理包含文字或数字的图像（如 TextVQA 基准）时这种提升尤为显著。

由于文字往往分布在画面的各个角落，任何位置偏见都会导致关键字符的丢失。

去偏后的模型能够像扫描仪一样公平地检阅图像中的每一个字符，从而在大幅精简数据量的同时，不丢失任何一个关键的数字。

这项研究告诉我们，大模型的认知蓝图里也充满了由于工程妥协而产生的裂缝。

通过这种即插即用的去偏手段，可以提升现有模型的运行效率，消除注意力机制中长期被忽视的偏置问题。

这种轻量化且高回报的优化路径，为未来更智能、更高效的多模态系统开发提供了一个极具参考价值的范本。
